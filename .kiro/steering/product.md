# Tashreef Product Context

## What This Application Does

Tashreef generates e-invitation cards with mathematical patterns inspired by ancient architectural designs (Islamic geometry, fractals, parametric curves). The system uses **mathematical computation**, NOT generative AI images, to create precise SVG geometric patterns.

## Core Approach: AI + Math Hybrid

```
Natural Language Prompt → LLM (Gemini 2.0 Flash) → JSON Parameters → Mathematical Engine → SVG Pattern
```

The LLM translates user intent into mathematical parameters. The actual pattern is generated by deterministic algorithms.

## Available Pattern Engines

1. **Parametric Equations** (`parametric_engine/`) - Spirograph-style curves using parametric functions
2. **L-Systems** (`fractal_engine/`) - Recursive fractal patterns via turtle graphics
3. **Tessellations** (`tessellation_engine/`) - Islamic geometric tiling patterns

## Generation Flow

1. User provides natural language prompt (e.g., "elegant floral pattern for wedding invitation")
2. LLM interprets prompt and outputs JSON with mathematical parameters
3. Engine generates base pattern SVG from parameters (pure math, deterministic)
4. Processor creates tiled/repeating backgrounds for card layout
5. Final SVG card combines pattern background with content placeholders

## Critical Constraints (Never Violate)

- **Output is ALWAYS SVG** - vector graphics only, never raster images (PNG/JPG)
- **Patterns are mathematically generated** - engines use equations, NOT AI image generation
- **No generative AI for images** - LLM only generates parameters, not visual content
- **Customization happens client-side** - dimensions, colors, text content modified in frontend
- **No external image assets** - pure geometric computation, no image dependencies

## Key Product Principles

- Precision over approximation (mathematical exactness)
- Scalability (SVG scales infinitely without quality loss)
- Cultural authenticity (patterns inspired by historical designs)
- Customizability (users can modify colors, text, dimensions)
